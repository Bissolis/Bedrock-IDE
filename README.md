# Bem-vindo ao Bedrock IDE

O **Bedrock IDE** foi criado com o objetivo de ajudar iniciantes na criação de scripts para **Minecraft Bedrock Edition**, tornando o aprendizado da API mais simples e intuitivo por meio de uma interface visual.

## Como o aplicativo é organizado

O aplicativo é dividido em **4 áreas principais**:

### 📌 1º Bloco — Eventos

Nesta área ficam todos os eventos da API que podem ser utilizados em seus scripts.

### 🧩 2º Bloco — Editor Visual

Aqui você encontrará os blocos de código. Você pode arrastá-los e organizá-los da forma que preferir para montar sua lógica.

### 📄 3º Bloco — Código JavaScript

Depois de montar sua lógica, os blocos serão convertidos automaticamente para **JavaScript**, permitindo visualizar o código gerado.

### 💻 4º Bloco — Terminal

O terminal exibirá mensagens, erros e avisos relacionados ao seu script, ajudando a identificar e corrigir problemas durante o desenvolvimento.

## Avisos

O aplicativo também possui dicas e mensagens distribuídas pela interface para orientar você durante o uso e facilitar o aprendizado da API.

Esperamos que o **Bedrock IDE** torne sua jornada na criação de addons para Minecraft Bedrock muito mais fácil e divertida. 🚀
