# Bedrock IDE
movido pela pela motivação de construir uma IDE, decidi desenvolver uma para ajudar os iniciantes na scriptização do minecraft bedrock
## apresentação da IDE
essa IDE possui algumas partes, sendo elas:
<br>
<img src="./bedrock.png" alt="bedrock" width="50"/> - O icone do app, leva direto para este github ao clicar
<br>
<br>
<img src="./book.png" alt="book" width="50"/> - icone que se refere a pagina inicial, nela possui a tela de boas vindas e algumas informações
<br>
<br>
<img src="./command-block.png" alt="command-block" width="50"/> - esse icone se refere ao editor, ainda pretendo ver se usarei o botão de iniciar ou se deixarei esse icone.
<br>
<br>
<img src="./recipes.png" alt="recipes" width="50"/> - esse icone levará você para a documentação explicando a construção dos scripts
<br>
<br>
<img src="./gift.png" alt="gift" width="50"/> - ao clicar nesse icone, você será levado a uma pagina de agradecimentos e um menu especial, entre para ver ;)

<br>
